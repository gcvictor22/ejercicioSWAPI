¿Qué es esta página?

¿Tiene problemas para recordar nombres, pero eres excelente para recordar caras? Nosotros también. Es por eso que hemos creado esta aplicación web. Todo se basa en colocar el ratón sobre una tarjeta, por lo que si está interesado en ver todos los personajes de las películas de Star Wars, haga clic en la categoría de personajes. Después de eso, simplemente apunte a cualquier imagen para obtener más información sobre las características físicas de ese personaje, películas relacionadas, planetas, vehículos, etc.

¿Por qué hay información desconocida?

Esta aplicación utiliza la API de Star Wars, por lo que la información que se muestra en esta aplicación se actualizará a medida que se actualice la API. Haremos otras actualizaciones aquí y allá, así que vuelva a consultar con frecuencia.

¿Has encontrado algún bug o una idea para mejorar?

Me la pela, este ejercicio no esta pensado para ver como trabajamos el código, esta pensado para ver como trabajamos con el compañero, con git sobre todo. Si hay cualquier fallo es cosa del codepen.io. De ahi hemos sacado la página de error 404 y esta página.
